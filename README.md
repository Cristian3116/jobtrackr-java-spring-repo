**JobTrackr** is a web application for managing job applications, built with **Java**, **Spring Boot**, and **Thymeleaf**.  
This project demonstrates **full-stack Java skills** and best practices, ideal for a portfolio.

---

## Features

- User authentication with roles (USER/Admin)  
- Create, edit, and delete job applications  
- Dashboard with statistics and status (Applied, Interview, Offer, Rejected)  
- Modern UI with **Thymeleaf** and **Bootstrap 5**  
- Data validation and relationships between entities (User â†” JobApplication)  
- MySQL database for local development
